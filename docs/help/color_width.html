Changing color and width of the pen


It is possible to change pen color and pen width by clicking 'Color' and 'Width' buttons corresponing. The default value for color is blue (0 85 255) and for width is 2 px.

To change pen color select preffered one in opened dialog and click 'OK' button.

To change pen width type in needed value in opened dialog and click 'OK' button.


Hotkey for opening changing color dialog: Shift+C

Hotkey for opening changing width dialog: Shift+W
